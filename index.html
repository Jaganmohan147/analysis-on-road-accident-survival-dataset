Analysis on Road     accident Survival     Dataset

Abstract

Road accidents cause injuries and deaths worldwide. Some people survive, while others do not. This project studies a Road Accident Survival Dataset using Pandas Profiling to find patterns in accident data. Factors like speed, weather, road conditions, and medical response time affect survival rates. The goal is to analyze data and understand how these factors influence accident outcomes.

Introduction

Accidents happen due to driver mistakes, poor road conditions, and weather effects. Some accidents are minor, while others are fatal. Quick medical help can improve survival chances. This project uses Pandas Profiling to quickly analyze accident data. It helps in identifying key factors that affect survival rates and improving road safety measures. 

Literature Survey

Many studies analyze accident causes and survival rates. Research shows that high speed, bad weather, and slow medical response reduce survival chances. Some studies focus on seatbelt usage and airbags, which improve safety. Machine learning is also used to predict accident outcomes. Pandas Profiling helps in automating data analysis and generating quick reports.

EDA & Project Overview

Exploratory Data Analysis (EDA) Using Pandas Profiling
EDA helps in understanding accident patterns. Pandas Profiling is used to generate a detailed report, including:

• Missing values – Identifying gaps in the dataset.

• Correlation analysis – Checking how factors like speed and weather affect survival.

• Summary statistics – Finding average accident severity, response time, and survival rates.

• Visualizations – Charts for accident trends, survival rates, and high-risk factors.
